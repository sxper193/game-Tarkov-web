# 美术资源需求清单

请生成以下图片，并保存为 **PNG 格式**（带透明背景）。
生成后，请将所有图片放入项目根目录下的 `public/assets/` 文件夹中（如果没有该文件夹请新建）。

## 1. 地面贴图 (Seamless Tiles)
*要求：正方形，无缝拼接（Seamless），俯视视角。*

| 文件名 | 描述 | 建议尺寸 |
| :--- | :--- | :--- |
| **`tile_grass.png`** | **草地**。深绿色，逼真的草叶纹理，带有少许泥土或杂草细节。避免明显的重复图案。 | 512x512 |
| **`tile_road.png`** | **沥青路面**。深灰色，带有颗粒感，可能有细微裂缝。**不要**包含车道线（车道线我们用代码画或者单独贴图）。 | 512x512 |

## 2. 环境物体 (Environment Objects)
*要求：俯视视角 (Top-Down)，带透明背景，投影方向统一（建议无投影或柔和顶光）。*

| 文件名 | 描述 | 建议尺寸 |
| :--- | :--- | :--- |
| **`spr_tree.png`** | **大树**。圆形或不规则的树冠，深绿色，有体积感。 | 256x256 |
| **`spr_bush.png`** | **灌木丛**。不规则形状，绿色，用于点缀。 | 128x128 |
| **`spr_crate.png`** | **木箱/板条箱**。正方形，木质纹理，可能有金属包角。 | 128x128 |
| **`spr_roof.png`** | **房屋屋顶**。矩形，红瓦或深色瓦片，中间有屋脊。 | 512x384 |

## 3. 角色 (Characters)
*要求：俯视视角 (Top-Down)，头顶+肩膀，带透明背景。面向右侧 (Right) 或 上方 (Up) 均可（代码可调整）。*

| 文件名 | 描述 | 建议尺寸 |
| :--- | :--- | :--- |
| **`spr_player.png`** | **特种兵/玩家**。穿着战术背心，手持突击步枪。 | 128x128 |
| **`spr_enemy.png`** | **匪徒/敌人**。穿着便服或破旧装备，手持武器。 | 128x128 |

3.  将图片放入 `public/assets` 文件夹。
